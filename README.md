## How to Use This Repo

**For agents:** Read all `s*.md` files first for full project context. Identify which section(s) the change belongs to using the table above. Edit only those files. Commit with message format `§N: <short description>`. Only update this README if a section file is added or deprecated.

**For humans:** Navigate to the section file most relevant to your change. Each file is self-contained. Cross-references use `→ see §X.Y` notation pointing to the relevant file.
